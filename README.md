# 🎤 Conference Management System - SQL Project

## 📌 Project Overview
This project demonstrates the use of SQL to manage and analyze conference-related data. 
The database includes information about attendees, speakers, sessions, and registrations.

## 🗄 Database Tables
- Attendees
- Speakers
- Sessions
- Registrations
- Conference Details

## 🔎 SQL Concepts Used
- SELECT statements
- WHERE conditions
- GROUP BY & HAVING
- JOIN operations
- Aggregate Functions (COUNT, SUM, AVG)
- Subqueries
- ORDER BY

## 📊 Key Analysis Performed
- Total number of attendees
- Session-wise registration count
- Speaker-session mapping
- Most popular sessions
- Attendance statistics

## 🛠 Tools Used
- SQL
- Database Management System (MySQL/PostgreSQL)

## 🎯 Objective
To design and query a relational database system for efficiently managing conference operations and extracting meaningful insights.
